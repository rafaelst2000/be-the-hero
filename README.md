# Be The Hero
> Projeto desenvolvido durante a semana Omnistack da Rocketseat :) 


# Tecnologias Abordadas:
- NodeJS
- React
- React Native
- SQL


# Executando o Projeto

- Backend:
> 1. $cd backend
> 2. $npm install
> 3. $npm start

- Frontend:
> 1. $cd frontend
> 2. $npm install
> 3. $npm start
> 4. No seu navegador, acesse localhost:3000
